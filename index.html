Overboard
I created the game Overboard in collaboration with a 3D model artist. The game was developed using Unity for a game jam competition, where it was completed within 1 week. With the given theme "less is more", we came up with a game where the objective is to slow the sinking of your ship by throwing your cargo overboard, whilst a pesky squid returns it.

Abyssal
The game Abyssal was created with unity in collaboration with a 2D artist. The game was developed in 1 week for a game jam competition, where it recieved 2nd place. With the theme "Attraction and Repulsion", I created an endless survival game where the main weapon had boomerang characteristics. I went for a simple concept, polishing the game with a lot of finer detail.

Shape Shifted
I created the game Shape Shifted entirely myself, using Unity. The game was developed in 3 days for a game jam competition, where it recieved 5th place. With the theme "teleportation", I created an endless runner where the player teleports around to avoid danger. The game also includes a coin/shop system for character customization.

Tappy Tile
Tappy tile is my released mobile game I created myself, using Unity. It took approx 1 week to finish, and was one of my first unity games that helped me develop my skills and passion for making games. Tappy Tile is a hyper-casual game where you tap the tiles that appear in order, aiming for a highscore on the global leaderboards.
